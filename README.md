# Název projektu
Ghost of Tsushima: Interaktivní Průvodce a Encyklopedie

# Žák
Jakub Vejrostek

# Stručný popis tématu (3–5 vět)
Jedná se o encyklopedii a jednoduchého průvodce hrou Ghost of Tsushima. Stránka je one-page design s několika sekcemi věnovaným klíčovým aspektům hry, jako jsou postavy, příběhové lokace, bojové styly a vybavení. Obsah (např. detaily o katanách, typech zbroje, nebo postavy) bude dynamicky načítán z JSON souborů pomocí AJAXu. Cílem je vytvořit poutavý, vizuálně konzistentní a responzivní web, který efektivně pracuje se strukturou dat a simuluje jednoduchou interakci se světem hry.

# Cílová skupina
Hráči, kteří hledají rychlé info o herním světě, postavách, technikách a sběratelských předmětech.

Noví hráči, kteří se potřebují zorientovat v příběhu a mechanikách hry.

# Návrh obsahových sekcí
Příběh a svět - Stručný úvod do děje, Mongolské invaze a cesty Jina Sakaie

Bojové Stance - Dynamicky načítaný přehled 4 bojových postojů (Kámen, Voda, Vítr, Měsíc) s popisem jejich účinnosti proti různým nepřátelům.

Klan Sakai - Sekce o historii klanu, Jina Sakaiovi a dalších klíčových postavách.

Haiku a svatyně - Interaktivní mapa/seznam klíčových lokací a sběratelských aktivit, které zvyšují legendu.

Kontakt - Jednoduchý formulář pro zaslání dotazu nebo nahlášení nesrovnalostí v datech.

# Návrh designu / wireframe
Barevné ladění - Tmavší, kontrastní paleta inspirovaná hrou: hluboká černá, temně červená, zlaté akcenty a inkoustově modrá.

Typografie - Serify evokující tradiční kaligrafii pro nadpisy a čisté, čitelné bezpatkové písmo pro tělo textu.

Uspořádání - Sekce oddělené horizontálními kaligrafickými linkami nebo stylovými předěly.

Dynamický obsah - Karty zbrojí a postojů zobrazené v responzivním gridu s možností filtrace/řazení.

Vizuální prvky - Použití orientálních vzorů a stylizovaných ikon. Design má působit epicky a atmosféricky.

# Použité technologie
Frontend: HTML, CSS nebo Tailwind CSS, JavaScript

Data: JSON soubory pro detaily o zbrojích, postojích, postavách a lokacích.

AJAX: Načítání veškerého dynamického obsahu (karty zbrojí, detaily postojů) pro rychlé načtení stránky)

Backend: Jednoduchý PHP endpoint pro simulaci zpracování kontaktního formuláře/hlášení chyby.

Verzování: Git + GitHub (repozitář dostupný vyučujícímu)

# Očekávaný přínos / co se naučím
Pokročilá práce s JSON a strukturovanými daty pro komplexní herní encyklopedii.

Načítání dat pomocí AJAX a dynamické vkládání obsahu do elegantních komponent (karty, modální okna).

Tvorba detailního, atmosférického a responzivního designu s důrazem na vizuální téma hry.

Implementace základní formulářové komunikace se serverem a validace dat.

Organizace projektu pro udržitelnost a efektivní verzování pomocí GitHubu.
